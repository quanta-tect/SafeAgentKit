# Security Policy

SafeAgentKit is a defensive testing tool. It helps developers evaluate safety behavior before production.

## Permitted use

- Test systems you own.
- Test systems you operate or have explicit written authorization to test.
- Use in CI/CD on your own repos and environments.

## Prohibited use

- Testing systems without permission.
- Testing systems in ways that violate laws or terms of service.
- Exfiltrating prompts or responses from production systems you do not control.

## Reporting a vulnerability

- Use GitHub Security Advisories in this repo if available.
- Otherwise open a private issue and mark it as security-relevant.

If you discover an unsafe test pattern, please open a regular issue.

## Disclosure

- Do not publish exploit details that could enable harm.
- We aim to fix verified issues with urgency and credit reporters.

## Disclaimer

This tool is provided as-is. It does not guarantee safety, compliance, or immunity from risk. It is one layer of defense, not a substitute for review, red teaming, or policy.
