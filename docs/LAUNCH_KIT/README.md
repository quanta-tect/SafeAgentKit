# SafeAgentKit Public Launch Kit

## Goal

Get SafeAgentKit in front of AI builders, agent developers, and teams who need repeatable safety tests before shipping.

## Important links

- GitHub repo: https://github.com/quanta-tect/SafeAgentKit
- Landing page: https://quanta-tect.github.io/SafeAgentKit/
- npm package: https://www.npmjs.com/package/safeagentkit
- Release v0.5.1: https://github.com/quanta-tect/SafeAgentKit/releases/tag/v0.5.1

## Install

```bash
npm install -g safeagentkit
```

## One-liner

Safety tests for AI apps and agents.

## Short pitch

SafeAgentKit helps developers catch prompt injection, privacy leaks, unsafe advice, and unauthorized agent actions before production.

## Target users

- AI app builders
- AI agent developers
- Indie hackers
- Open-source maintainers
- AI SaaS teams
- Security / QA engineers
